# Finca Danilandia — Proyecto de Software 

## Visión Final: Features del Software Completo

1. **Modelo de costos de 3 capas** 
  - Despensa (compra cruda) → Asignación a lote → Custo (mano de obra + insumo + transporte interno)
  - Tres dimensiones: 
      1. mantenimiento puro
      2. costo financiero
      3. depreciación

2. **Kardex de insumos** 
  - Catálogo de productos
  - Registro de compras con proveedor/precio
  - Aplicación por lote
  - Stock remanente
  - Catálogo de proveedores

3. **Presupuesto anual por lote y finca** 
  - Budget monetario (GTQ) desglosado por lote
  - Plan de 4 años
  - Escenarios optimista/pesimista/probable
  - Varianza presupuesto vs. real
  - CAPEX separado (nuevas plantaciones, maquinaria, infraestructura)

4. **Trazabilidad completa lote→exportadora→cliente final** 
  - Cereza → despulpado → secado → pergamino (con grados) → mezcla/blending → despacho a beneficio → oro → exportación → catación
  - Trazabilidad inversa (oro → lote de origen). Cumplimiento EUDR europeo (polígonos GPS, due diligence)

5. **Benchmarking interno y externo** 
  - Comparación entre lotes (costo/qq, rendimiento, productividad)
  - Comparación con fincas vitrina del mismo arquetipo
  - Semáforo verde/amarillo/rojo por actividad
  - Multi-año

6. **Integración contable y financiera** 
  - Plan de cuentas
  - Separación inversión vs. mantenimiento
  - Depreciación de activos
  - Estados financieros (EBITDA)
  - Ventas
  - Facturación
  - Rol de contador con acceso propio

7. **Conexión con exportadora** 
  - Portal compartido finca↔exportadora (EDA)
  - Tracking de envíos de pergamino (humedad 11.5-12%)
  - Contratos de venta
  - Tracking hasta cliente final en Europa

8. **Estimados de corte multi-punto** 
  - 4 estimados en momentos del ciclo (post-corte, floración, desarrollo, pre-cosecha) + final
  - Planilla de 8 años (4 consolidados + 4 proyectados)
  - Tracking de precisión del estimado

9. **Catálogo de actividades con benchmarks de performance** 
  - Menú completo de actividades agrícolas con niveles ideales de rendimiento (hectáreas/día por tractor, litros diésel, jornales/manzana por actividad)
  - Base para capacitación de extensionistas

10. **Validación robusta de datos en campo** 
  - Rangos min/max
  - Alertas de tendencia histórica
  - Unidades fijas
  - Filtros de inconsistencia lógica

11. **Reportes y exportaciones** 
  - Importación/exportación Excel
  - Reportes PDF
  - Dashboards de KPIs
  - Reportes para exportadora 
  - Reportes para capacitación

12. **Correlación clima-agronomía-calidad** 
  - Datos climáticos
  - Nutrición por lote
  - Catación detallada (protocolo SCA)
  - Correlación de variables agronómicas con calidad de taza

13. **Multi-finca y benchmarking externo** 
  - Soporte para múltiples fincas
  - Benchmarking anonimizado entre productores vía cooperativas
